Kirigami has a built in architecture to manage various aspects of the styling of the application, such as colors, widget style (Breeze, Material and so on) icon sets, typography and the general organization, and UI navigation within the application.
See the Human Interface Guidelines about the :doc:`style in the HIG <hig:style/index>` for more details.
