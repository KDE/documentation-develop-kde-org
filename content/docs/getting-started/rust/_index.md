Rust has become a popular programming language with many desirable features for many fields of development, but it has yet to gain more reach for desktop applications.

QML offers an intuitive way to create user interfaces.
[Kirigami](https://develop.kde.org/docs/getting-started/kirigami/) extends QML
to provide useful UI components and it implements UI/UX patterns for mobile and
desktop.

We will cover this gap by teaching you how to make desktop applications using Rust and QML, with the help of [cxx-qt](https://kdab.github.io/cxx-qt/book/), a project that lets you write Rust and use C++ libraries like Qt with idiomatic Rust.
